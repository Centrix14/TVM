10 steps to the program on nom!

Step 1

Go to the Telegram channel TVM/Project (https://t.me/tvm_project) and find the last post marked {Release}, follow the link and download the archive with the files.


Step 2

Unpack the archive in some directory, and remember the path where unpacked. For Example: D:\


Step 3

Then, add an environment variable that resembles the following:..\TVM\a TVM\src. In our primerima write: D:\TVM\TVM\src.


Step 4

Open the console, go to the directory ...\TVM\a TVM\src


Step 5

Type the following command into the console: gcc nom.c -o nom.exe


Step 6

And after the previous command completes, enter the second: gcc ncprun.c -o run.exe


Step 7

Now you can exit this directory and create a new file. Well, if you want to stay in the current one, we recommend creating a separate directory. The file must have the extension (.nom), open it and write down the following: 

.main 
put 72 _cwr 
put 101 _cwr 
put 108 _cwr
put 108 _cwr
put 111 _cwr
put 32 _cwr

put 119 _cwr
put 111 _cwr
put 114 _cwr
put 108 _cwr
put 100 _cwr
put 33 _cwr
.end


Step 8

Now type the following into the console: nom your_file name.nom new_file.ncp


Step 9

Now enter some more: run new_file.ncp


Step 10

If you did everything right, you saw the inscription "Hello world!". Congratulations you can continue to study, good luck!
